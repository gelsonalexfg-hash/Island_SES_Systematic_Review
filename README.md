# Island SES Systematic Review

This repository accompanies the manuscript:

**Socio-ecological Systems in Small Island Contexts: A Systematic Review of Advances Reconciling Human Activities with Biodiversity Conservation**

## Repository contents

This repository provides the analytical materials supporting the systematic review.

### Data

- Replica.csv (structured extraction dataset)

### Supplementary Materials

- Supplementary Material S1 – Analytical Codebook and Coding Protocol
- Supplementary Table Sx – Study Classification Matrix

### Analytical notebooks

- 01_Methodological_Classification.ipynb
- 02_Analytical_Frameworks.ipynb
- 03_Socioecological_Interactions.ipynb
- 04_Social_Ecological_Variables.ipynb
- 05_Thematic_Synthesis.ipynb

### Python scripts

Equivalent Python scripts are provided for each notebook.

## Reproducibility

The repository contains the extraction dataset, analytical codebook, study classification matrix, analytical notebooks, and scripts used to generate the figures and tables reported in the manuscript.

Full-text articles are not distributed because they remain protected by copyright.
